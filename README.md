## ğŸš€ Quick Start

### Prerequisites

- Node.js (v18 atau lebih baru)
- npm atau yarn

### Instalasi

1. **Clone repository**

   ```bash
   git clone https://github.com/uneeddeveloper/edugate.git
   cd edugate
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Run development server**

   ```bash
   npm run dev
   ```

   Aplikasi akan berjalan di `http://localhost:3000`

4. **Build untuk production**
   ```bash
   npm run build
   ```
